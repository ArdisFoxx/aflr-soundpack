# AFLR Soundpack

The recommended companion soundpack for **[ArdisFoxx's Lewd RPG](https://github.com/ArdisFoxx/ardisfoxxs-lewd-rpg)** on Foundry VTT.

A curated selection of voices and SFX. The download covers AFLP's full audio feature set.

## Install

In Foundry: **Add-on Modules -> Install Module**, paste this manifest URL, and click Install:

```
https://github.com/ArdisFoxx/aflr-soundpack/releases/latest/download/module.json
```

Then enable **AFLR Soundpack** under Manage Modules. AFLP detects it automatically - there is no path to configure. Turn on Voice and/or Ambient SFX in the AFLP settings and assign a voice profile per actor from their AFLP sheet.

Manual install: download `module.zip` from the [latest release](https://github.com/ArdisFoxx/aflr-soundpack/releases/latest), unzip into `Data/modules/`, and restart Foundry.

## Requires

- **ArdisFoxx's Lewd RPG** (`ardisfoxxs-lewd-rpg`). The soundpack has no effect on its own.

## What's inside

- **Voice profiles** covering the full event set (moans by arousal tier, climax, edge, advance, struggle, defeat, cumflation, oral), with female, male, and creature voices.
- **Ambient SFX** for the categories the engine uses: positional plaps, oral (gluk), fingering (schlick), and the cumflation stings.

If a voice profile is missing a specific event category, AFLR falls back gracefully, so every profile still works.
